# MC Skin Dataset Manager (V1.0)

## 项目简介
本项目是一个专为 Minecraft 皮肤绘制、游戏角色管线以及 AI 图像生成模型（如 LoRA）提供数据整理与标注的桌面端工具。
它旨在解决海量图像分类、版权追踪、元数据（Prompt）打标以及项目级文件管理的问题。

## 核心特性
* **无感数据管线**：将原始图片放入 `staging` 文件夹，一键快速分类并自动归档至结构化的项目文件夹。
* **多模态元数据存储**：摒弃传统零散的 txt，采用全局强健的 `metadata.jsonl` 进行数据统一管理。
* **双轨制打标系统**：
    * **主界面捷径**：快速勾选版权、AI 生成标识与作者来源，与后台数据实时联动。
    * **深度标注台 (Deep Annotation Desk)**：独立的沉浸式窗口，用于精细调节正/反向提示词，并预留了详细物理特征的扩充接口。
* **所见即所得的头像裁剪**：内置原生截图遮罩模块，一键生成标准 256x256 项目缩略图。
* **跨平台 CMake 架构**：彻底摆脱对特定 IDE 的依赖，基于 C++17 与 Qt 6 构建。

## 目录结构
* `DatasetCore/` - 底层数据引擎（静态库），负责极其稳定的文件读写与 JSON 序列化。
* `MCSKinApp/` - 核心图形化界面应用。
    * `MCSkinApp.ui` - 管线分拣主界面
    * `AnnotationWindow.ui` - 深度特征标注台
    * `GalleryWindow.ui` - 全局项目画廊与检索
* `Workspace/` (运行时自动生成)
    * `staging/` - 原始图片放置区
    * `Minecraft_Skin_Dataset/` - 最终的有序数据集库

## 快速开始
1. 启动 `MCSKinApp.exe`。
2. 点击右上角选择一个空白文件夹作为 Workspace。
3. 将杂乱的参考图、皮肤源文件拖入自动生成的 `staging` 文件夹。
4. 点击“新建空白项目”，开始你的高效分拣与标注之旅！
