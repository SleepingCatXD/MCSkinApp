MC Skin Dataset Manager (内测版) - 基本操作指南
欢迎参与本项目的数据标注内测！本软件专为高效整理、分类和标注角色数据集而设计。请按照以下工作流进行操作：

一、 准备工作

解压收到的压缩包，不要移动里面的任何 .dll 文件。

在软件所在目录下，你会看到一个名为 staging 的文件夹（如果没有请手动新建）。

将你需要处理的所有原始图片、皮肤文件等直接拖入 staging 文件夹中。

双击运行 MCSkinApp.exe 启动程序。

二、 核心工作流：建立你的第一个项目
在我们的系统中，一个“项目 (Project)”代表一个独立的角色或资产集合。所有的相关图片（原画、皮肤、AI生成图等）都必须归档到对应的项目 ID 中。

选择工作区：点击左上角的“选择工作区”，选中软件所在的根目录。

新建项目：点击“新建空白项目”。此时左侧栏会亮起，显示为你分配的专属项目 ID（如 000001）。

分发图片：

观察中间显示的大图（来自你的 staging 文件夹）。

点击右侧的“图片分类”按钮（如“目标角色”、“参考风格”、“人工皮肤完稿”等）。

该图片会瞬间被重命名并移动到左侧的项目列表中。中间会自动显示下一张待处理图片。

截取项目头像：

在左侧文件列表中，点击你刚刚分发进去的某张图片（例如目标角色图），它会在中间放大显示。

点击下方的 “截取当前图为头像”（此时图片变暗，出现虚线框）。

按住鼠标左键拖拽框选面部，滚动鼠标滚轮可缩放方框大小。

调整完毕后，再次点击该按钮确认。左上角将永久保留该角色的精美头像。

添加提示词 (Prompt)：

点击“将此文本设为Prompt”，在弹出的窗口中输入该角色的核心特征。例如，你可以输入：“white hair with blue highlights, black horns, heterochromia (left eye blue, right eye purple)” 等关键细节。确认后，这些信息会永久保存在该项目中。

三、 错误回退与画廊管理

放错图片怎么办？ 在左侧列表中选中放错的图片，点击红色的 “将此文件退回 Staging”，它就会立刻回到待处理队列中。

如何查看所有项目？ 点击顶部菜单栏的 View -> Gallery Database，或者直接点击左上角的头像框。

画廊模式：在弹出的画廊中，你可以看到所有已经建立的头像。双击任意头像，即可瞬间恢复到该项目的工作状态，继续为它添加图片或修改提示词。如果有彻底作废的项目，也可以在画廊中选中并安全删除。
